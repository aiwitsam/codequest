How to Update Your System

1

Quick Update (Safe)

Updates all packages to their latest versions. This is safe to run anytime.

sudo apt update && sudo apt upgrade -y

This downloads the latest package lists and installs available updates.

2

Full Update (Thorough)

Also installs updates that need new packages or remove old ones.

sudo apt update && sudo apt full-upgrade -y

Use this when the quick update says packages were "held back".

3

Clean Up

Remove old packages that are no longer needed.

sudo apt autoremove -y && sudo apt autoclean

Frees up disk space by removing leftover dependencies.

?

What Does "sudo" Mean?

sudo = "Super User DO" — it runs the command as an administrator. Your system will ask for your password. This is normal and required for installing updates.
